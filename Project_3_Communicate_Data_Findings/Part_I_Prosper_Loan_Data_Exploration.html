Introduction¶

This project explores loan data from Prosper, which contains 113,937 loans with 81 variables on each loan. Some of the loan features are listed and briefly defined below. The other loan variables will be introduced during the data exploration:

  • LoanNumber: Unique numeric value associated with the loan.
  • CreditGrade: The Credit rating that was assigned at the time the listing went live.
  • Term: The length of the loan expressed in months.
  • LoanStatus: The current status of the loan: Cancelled, Chargedoff, Completed, Current, Defaulted, FinalPaymentInProgress, PastDue.
  • BorrowerRate: The Borrower's interest rate for this loan.
  • LoanOriginalAmount: The origination amount of the loan.

What is the structure of your dataset?¶

The Prosper loan dataset contains 113937 loans, with 81 features recorded for each loan. Most loan features contain numeric values. For example, the variables Term, BorrowerRate and LoanOriginalAmount are numeric variables while CreditGrade and LoanStatus are string varibles containing categorical variables. Other variables such IsBorrowerHomeowner and IncomeVerifiable contain boolean values.

What is/are the main feature(s) of interest in your dataset?¶

In this project, I am interested in finding out what features influence the borrower's interest rate for the loan and whether these features are best for predicting the borrower's interest rate for the loan.

What features in the dataset do you think will help support your investigation into your feature(s) of interest?¶

My expectation is that the borrower's interest rate will be influenced by features such as the loan term, the loan amount and the Prosper score. I also expect the number of current delinquencies to have an influence on the borrower's interest rate.

Before beginning my exploration, I first created a copy of the dataset, dropped all columns not relevant to the exploration. Then I cleaned the data by dropping null values and incorrect data. During univariate exploration, I plotted graphs for each individual variable in order to check for the distribution of each variable.

In bivariate eploraionn I observed a positive relatioship between the borrower rate and the lender's yield, and a negative relationship between borrower rate and the Prosper score.

In multivariate exploration, I plotted two scatter plots between the borrower rate, lender yield and the categorical variables incomeVerifiable and IsBorrowerHomeOwner. Despite seeming to fall on the trend line between borrower rate and lender yield, these categorical variables do not appear to influence the borrower rate.
